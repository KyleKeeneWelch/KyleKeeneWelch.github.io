Summary

This project was my first attempt at building a simple React application using the teachings from The Odin Project. It was a simple CV application that allows the user to enter in some details for sections such as general, education, and experience, and printed this information back to the user as their formatted "CV". The simplistic nature enabled me to get a grip with how components and hooks work in React and served as a starting point for building my skills in React to be used in my final year project.

Features

  • Show and Hide Sections - Enables the user to toggle between displaying a section and hiding.
  • Add Entries - Users can click the add button to insert more fields to enter more information on a specific section.
  • Produce a compiled "CV" with the entered information - Pressing show on your produced CV will display the information back to the user.

Implementation

Nesting Components

The App component provides the layout of the application and nests components within to serve as the sections for the user to enter information or display their entered information. Each referenced component is passed properties that can be used within the component and updated as state changes.

UseState Hook

The UseState hook is used to maintain specific values between renders of a component. It also triggers a render when changed. We pass the variable and setter function to components that we want to access and change this state. When interactions with the user cause the state to change, all the components that utilize the state will re-render to reflect the changes. The provided code stores state as objects relating to the fields in components and the display of sections. Changes to fields or clicking a show / hide button will alter the state and cause components to re-render.

Data Mapping

The Map function is used to produce a DOM element for each entry in the passed data. In this case, we used Map to loop through each place of education the user has entered in the application and incremented this value to prepare fields for a new entry.
